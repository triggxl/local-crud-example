      - <form>

  2) Fill out the form
    i) controlled inputs
      - state, onChange, value
  3) Submit information
    i) store information
      - state (App)

-Read

-Update

    i) controlled inputs
      - state, onChange, value
  3) Submit new information
    i) find specific item in list
    ii) replace with user input



-Delete

  2) Filter out the bugs list using the id of the bug clicked
